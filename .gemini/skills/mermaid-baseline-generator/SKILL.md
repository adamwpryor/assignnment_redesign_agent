# `mermaid-baseline-generator` Skill

This skill is used by the `resilient-designer` agent when constructing assignments that require visual framework deliverables.

## Usage

When an assignment requires a "Visual Framework Defense," the Resilient Designer calls this tool to quickly draft the baseline Mermaid.js code that serves as the "AI's first pass" for the student to critique and intentionally break.
